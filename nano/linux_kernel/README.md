리눅스 커널을 공부해약겠다고 생각한 것은 현업에 들어오고 1개월 만이었다.
사실 학부 과정에서 운영체제 수업을 듣긴했으나 순식간에 증발하는 뇌 때문에 기억이 희미하다.

서비스를 개발하고 운영해야하는 DevOps 처지가 되자, 자연스래 시스템 엔지니어링 지식이 필요해졌다.

정말 기본적인 것부터 모른다는 사실에 충격을 받았고, 이에 대한 공부를 위해 일단 워밍업으로 한것이 tcp공부이다
network 디렉터리에 tcp 공부 내용을 적어뒀지만 아직 빠진 지식이 많고 차차 공부해나갈 것이다.

이 디렉터리에는 'DevOps와 SE를 위한 리눅스 커널 이야기' 라는 책을  읽으면서 공부하는 내용을 정리할 것이다.
